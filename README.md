# MC-MODUTECH-WEB-SERVICE

This is the project of `mc.modutech.win` minecraft server's web site project.

## Prerequired

- docker
- node 17.8.0 이상

## 로컬에서 다운받아 실행 해보는 방법

- 도커로 실행 해보기

```bash
git clone https://github.com/GeunSam2/mc-modutech-web-service
cd mc-modutech-web-service
make build
make run
```

- npm 실행

```bash
npm start
```
